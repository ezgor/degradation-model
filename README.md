# degradation-model

##Quick start
1. Clon repo
``` git clone https://github.com/ezgor/degradation-model
cd degradation-model
```
2. Do 
``` python3 run.py ./saved_weights/models/mod_*  [inputs folder path] [output folder path] [size of block to split the image; default 256] ```
